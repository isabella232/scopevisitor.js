scopevisitor
============

scopevisitor is a node.js package that visits JavaScript scopes created by
[marijnh/tern](https://github.com/marijnh/tern)'s [inference
engine](http://ternjs.net/doc/manual.html#infer).


Running tests
=============

Run `npm test`.
